Instructions

First time competing? Forgotten what to do? Follow these instructions and you'll be fine.
  1. Solve the prescribed problems as fast as possible.
  2. Solve the problems in any order you like.
  3. Ensure your output matches the format given for each problem.

Rules

Breaking these rules will result in the competitor being disqualified and their prizes forfeit.
  1. You must work as an individual. Do not collude with others or work in teams.
  2. Use only documentation described below.
  3. A prepared programming template is allowed. You can't copy-paste algorithms though!
  4. Don't do anything that would provide an unfair advantage.

If you want to clarify these rules, ask within #competitions and @committee to get our attention.

Limits

  • The time limit is different for each problem.
    • Java solutions are allowed for 5 times of the time limits stated.
    • Python 3 solutions are allowed for 10 times of the time limited stated.
  • The memory limit for each problem is 512 MiB.
  • The source code limit is 128 KiB.

Compilation Specifications

C++
Version:gcc 6.3.0 20170516
Compilation:g++ -Wall -O2 -std=c++14 -static -pipe -DONLINE_JUDGE -DDOMJUDGE -o prog prog.cc
Execution:./prog < input > output

Java
Version:1.8.0_181
Compilation:javac -encoding UTF-8 -d . prog.java
Execution:java -client -Xss128m -DONLINE_JUDGE=1 -DDOMJUDGE=1 < input > output

Python 3
Version:3.5.3
Compilation:python3 -m py_compile prog.py3
(Just used to check for Compile Errors, not in the execution of the code)
Execution:python3 prog.py3 < input > output

C
Version:gcc 6.3.0 20170516
Compilation:gcc -x c -Wall -O2 -std=c11 -static -pipe -DONLINE_JUDGE -DDOMJUDGE -o prog prog.c -lm
Execution:./prog < input > output
